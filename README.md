# Yungg Star Graphics Website

Premium graphic design agency website — Accra, Ghana.

## Deploy to Netlify
1. Go to [netlify.com](https://netlify.com) and sign in.
2. Drag & drop this folder onto the Netlify dashboard.
3. Done — your site is live! ✅

## Deploy to Vercel
1. Go to [vercel.com](https://vercel.com) and sign in.
2. Click **Add New Project** → **Upload** → drag & drop this folder.
3. Click **Deploy**.
